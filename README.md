# FFD.jl
A Julia repository containing code for Free Form Deformation using B-splines.
The repository has the following structure

* FFD.jl
  * src
    * bounding_box.jl
    * b-splines.jl
    * control_point.jl
    * evaluations.jl
    * extra.jl
    * knot.jl
    * mapping_functions.jl
    * mapping.jl
    * param_functions.jl
    * plot.jl
    * span.jl
    * startup.jl
  * test
    * runtest.
  * README.md
